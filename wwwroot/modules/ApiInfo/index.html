API Info

For normal Timesheet API calls, the root is https://api.dovico.com/ and then you add the resource you want (e.g. employees/?version=7).

For modules, Timesheet will handle calling the API for you. You call a Timesheet root URI similar to how you call the normal API.

To get the timesheet root URI, a property of the parent iFrame is available as shown in the following code snippet:

const rootUri = window.frameElement.getAttribute("data-APIRequestURI");



Timesheet API documenation:
https://www.dovico.com/developer/API_doc/index.htm

In production, a form of this file will be needed but not the API credentials properties because Timesheet handles that for you.

The structure needed for the ModuleManifest.json file in this simulator:

{
"Name": "My module's name",
"MenuCaption": "Hello User Module",
"StartPageUri": "index.html",

"ApiConsumerSecret": "consumer secret from your Timesheet db",
"ApiDataAcessToken": "system or user data access token from your Timesheet db"
}
